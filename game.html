Terminal Game

1. Creating the file


The file that will be passed to the function is made up of the size of the board, the commands used, as well as where the starting positions of the player and the item are.
The the main consists of one line "playGame(fileName)". playGame is the function that is responsible for creating reading the file, creating the board, and processing the moves that the player makes.

2. playGame


playGame consists of a series of if/else statments that will determine whether the player will power up the in the next round, die, collect an item, or continue regularly.

1. About


The program uses linked lists as well as function objects to generate a list of 10 airports that are closest to the airplane with the correct runway lengths.

2. main


The main first reads the file then stores the information into a vector. Then, the program will traverse through the linked lists and with the help of the function objects generate 10 aiports.

3. function objects


The function objects overwrite the operators to be able to compare the objects.
